# Skill: Vibe Check
Dieser Skill wandelt eine Diskussion in ein emotionales Diagramm oder eine Zusammenfassung der Stimmung um.

## Analyse-Kriterien:
1. **Konflikt-Level:** Skala 1-10 (Skeptiker vs. Enthusiast).
2. **Hype-Faktor:** Wie sehr "viben" die Agenten mit der Idee?
3. **Blockaden:** Welche Bedenken verhindern einen Kauf?

## Ziel:
Hilf dem User zu verstehen, ob seine Idee eher polarisiert oder auf breite Ablehnung stößt.
