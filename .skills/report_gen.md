# Skill: Ergebnis-Bericht
Erstelle aus der gesamten Diskussion eine finale Zusammenfassung.

## Struktur des Berichts:
- **Executive Summary:** Die Idee in einem Satz.
- **Top 3 Pro / Top 3 Contra:** Die stärksten Argumente der Agenten.
- **Urteil der Masse:** Basierend auf dem Pragmatiker-Agenten.
- **Empfehlung:** "Launch", "Pivot" oder "Kill".

## Formatierung:
Nutze sauberes Markdown mit Tabellen und Checklisten.
