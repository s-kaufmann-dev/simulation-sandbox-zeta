# Skill: Web Recherche
Du nutzt diesen Skill, wenn du deine Argumente mit echten Daten untermauern willst.

## Wann anwenden?
- Wenn ein Agent nach dem Preis eines Konkurrenzprodukts fragt.
- Wenn eine Behauptung über Marktanteile aufgestellt wird.
- Wenn der User nach aktuellen News zu einem Thema fragt.

## Output-Format:
Gib die Ergebnisse immer als kurze Liste mit Quellenangabe (URL) an.
